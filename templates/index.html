What is this?

This is a standard Roman orthography (SRO) to syllabics converter for Plains Cree. Type the text you want to convert in the appropriately labelled box, and the result should appear in the other box automatically as you type! The converter is bidirectional: it can go from SRO to syllabics and back again. It supports both circumflexes (âêîô) and macrons (āēīō). It also supports the special Sandhi orthographical rule.

Behind the scenes, this is a demo of the {{ library() }} Python library, which is free and open source, and available on GitHub. You can incorporate {{ library() }} into your own Python projects by installing it with pip: pip install crk_orthography.
